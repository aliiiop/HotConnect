# HotConnect

Рофл-игра про Гошу, преподавательницу колледжа, с выборами, мини-играми, спецэффектами и боями в духе Undertale.

## Описание проекта

HotConnect - это интерактивная веб-игра, созданная с использованием современных технологий. Игра представляет собой визуальную новеллу с элементами RPG-боёв в стиле Undertale, где игрок управляет преподавательницей колледжа Гошей и проходит через различные испытания рабочего дня.

## Технологический стек

### Frontend
- **React 18.3.1** - библиотека для создания пользовательских интерфейсов

- **Vite 7.3.2** - современный инструмент сборки

### Инструменты разработки
- **ESLint** - линтер для проверки качества кода
- **Vite Plugin React** - плагин для поддержки React в Vite

### Хранение данных
- **localStorage** - для хранения данных пользователей и прогресса игры

## Установка и запуск

### Требования
- Node.js версии 16 или выше
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение откроется по адресу `http://localhost:3000`

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`

### Предварительный просмотр продакшен-сборки

```bash
npm run preview
```

## Использование

### Регистрация и вход

1. При первом запуске перейдите на страницу регистрации
2. Введите имя пользователя (минимум 3 символа)
3. Введите пароль (минимум 6 символов)
4. Подтвердите пароль
5. После регистрации вы автоматически войдёте в игру

### Игровой процесс

#### Сюжетные сцены
- Читайте текст истории
- Выбирайте один из предложенных вариантов
- Ваши выборы влияют на развитие сюжета

#### Боевые сцены
- **Удар** - попадите в зелёную зону для нанесения урона
- **Действие** - выполните мирное действие для успокоения врага
- **Предмет** - используйте предмет для восстановления здоровья
- Уклоняйтесь от снарядов с помощью WASD или стрелок

#### Мини-игры
- **Поймай журнал** - кликайте по летающим бумагам
- **Кофейный таймер** - останавливайте стрелку в зелёной зоне

### Управление

- **WASD** или **Стрелки** - движение в бою
- **Space** или **Enter** - остановка таймера
- **Esc** - сдаться в мини-игре
- **Мышь** - выбор действий и кнопок

Игра автоматически меняет тему в зависимости от текущей сцены:

- **dry** - кофейная тема
- **battle-*** - уникальные темы для каждого босса
- **mini-*** - темы для мини-игр
